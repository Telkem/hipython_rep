## 📈 항공사 지연 데이터 분석 프로젝트 요약 (Airline Delay Data Analysis Project Summary)

본 프로젝트는 항공 운항 지연의 주요 원인을 파악하고, 항공사별 지연 특성과 패턴을 비교 분석함으로써 **항공 지연 문제에 대한 데이터 기반 인사이트**를 도출하는 데 목적이 있습니다.
`pct_carrier_delay`, `pct_atc_delay`, `pct_weather_delay`와 같은 주요 지연 요인 데이터를 바탕으로 다음을 수행합니다:

* **결측치 처리 전략 수립** 및 비교

* **탐색적 데이터 분석(EDA)** > 변수 분포 및 이상치 탐색

* **항공사별 지연 패턴 비교**

* **상관관계 분석**

* **시각화를 통한 인사이트 도출**

**데이터 출처**: <https://www.kaggle.com/datasets/pitterman/airline-stats>

* 항공사별 지연율 데이터 (총 33,468건)

* 주요 컬럼:

  * `pct_carrier_delay`: 항공사 내부 사정으로 인한 지연 비율

  * `pct_atc_delay`: 항공 교통 통제로 인한 지연 비율

  * `pct_weather_delay`: 기상 조건으로 인한 지연 비율

  * `airline`: 항공사명

## 🚇 지하철 시간대별 승하차 비율 분석 프로젝트 요약 (Subway Time-based Ridership Analysis Project Summary)

본 프로젝트는 지하철역별 출퇴근 시간 승하차 패턴 분석을 통해 서울 시민의 주거지 및 업무지 공간 분포를 추정하고, 교통 혼잡 개선 방향을 제시합니다.

**주요 분석 내용**:
* **데이터 전처리**: 결측치 없는 2020년 1월 이후 출퇴근 시간대 데이터 활용.
* **EDA 결과**:
    * **주거지 특성 역**: 오전 승차, 저녁 하차 인원 많음 (예: 신림, 구로디지털단지, 서울대입구).
    * **업무지 특성 역**: 오전 하차, 저녁 승차 인원 많음 (예: 강남, 역삼, 삼성, 을지로입구).
    * 서울 서남부는 주거지, 도심 및 강남은 업무지 특성이 강함.
* **결론 및 제언**: 교통 시스템 효율성 극대화, 수요 관리 및 분산 전략, 환승 시스템 개선, 스마트 교통 시스템 도입 필요.

**기술 스택**: Python, Pandas, Matplotlib, Seaborn
