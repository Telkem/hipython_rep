## 🧾 개요 (Overview)
이 레포지토리는 **LG U+ WHY NOT SW** 캠프 7기의 수업 과정을 정리한 개인 포트폴리오입니다.<br>
Python 기초부터 데이터 분석 입문, 그리고 팀과 함께한 키오스크 실전 프로젝트까지 학습 및 실습한 내용을 담고 있습니다.<br>
Jupyter 노트북 기반 실습과 실제 콘솔 환경 프로젝트를 통해 Python 전반에 대한 이해도를 높였습니다.
